Geheime Terminal-Befehle und macOS Tweaks für den digitalen Widerstand.

SYSTEM SAFE

Hardware-Kurzprofil !

system_profiler SPHardwareDataType | grep -E "Model|Chip|Memory|Serial" [KOPIEREN]

Model, Chip, RAM und Seriennummer auf einen Blick — ohne Klicken durch Systeminfo.

SYSTEM SAFE

macOS Version

sw_vers [KOPIEREN]

ProductName, Version und Build. Schneller als jedes UI-Menü.

SYSTEM SAFE

CPU exakter Bezeichner

sysctl -n machdep.cpu.brand_string [KOPIEREN]

Zeigt den genauen CPU-String wenn jemand fragt und du nicht suchen willst.

NETZWERK SAFE

Netzwerkqualität testen

networkQuality [KOPIEREN]

Apples eigenes Speedtest-Tool seit macOS 12. Misst Upload, Download und Responsiveness (RPM).

NETZWERK SAFE

Alle offenen Ports

lsof -i -n -P | grep LISTEN [KOPIEREN]

Welcher Prozess hört auf welchem Port? Unverzichtbar wenn ein Port schon belegt ist.

DATEIEN SAFE

DS_Store aufräumen

find ~ -name '*.DS_Store' -delete 2>/dev/null && echo 'Aufgeräumt.' [KOPIEREN]

Löscht alle .DS_Store-Dateien im Home-Verzeichnis. Gut vor git-Commits.

PERFORMANCE SAFE

Top 15 CPU-Fresser

top -o cpu -n 15 -l 1 | tail -16 [KOPIEREN]

Einmaliger Snapshot der 15 CPU-intensivsten Prozesse.

TWEAKS SAFE

Dock: sofortiges Einblenden!

defaults write com.apple.dock autohide-delay -float 0 && defaults write com.apple.dock autohide-time-modifier -float 0.3 && killall Dock [KOPIEREN]

Entfernt die Verzögerung beim Dock-Einblenden. Sofort aktiv.

SICHERHEIT SAFE

SSH Key im Agent speichern

ssh-add --apple-use-keychain ~/.ssh/pfadzumkey [KOPIEREN]

fügt für die aktuelle session den key zum ageten hinzu, beim nächsten aufruf von ssh host wird der key austomatisch benutzt
